link

The link element is used to link other resources. The most common resource being CSS. One thing to note is that JavaScript is not linked using the link element, but rather script. Stylesheets (CSS) are loaded by providing the rel="stylesheet" attribute. Alternate stylesheets can also be loaded.

Our implementation of the link element uses the onload attribute, which basically calls a JavaScript function when the link is done loading.
