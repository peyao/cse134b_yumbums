meta

The meta element is typically used to provide metadata which do not apply to any other existing meta related tags, such as: base link script style title. It is placed within the head of the document.

In this page (and every other page here) the meta tag is used to declare the character encoding of the document using the charset attribute. Without this attribute, the characters on foreign pages may not appear correctly.
