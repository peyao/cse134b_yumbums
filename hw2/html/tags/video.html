4.7.6 - video

index

video (.webm)

The most basic video element implementation with no extra attributes.

video with autoplay muted controls (.webm)

This video element autoplays has controls and is muted. It uses the three attributes: autoplay controls muted.

video with controls resized loops poster (.webm)

This video element employs the controls attribute, giving the player some basic controls, is resized, loops, and has a poster.

The attributes used: controls width="848" height="480" loop poster="../../public/img/barbecue.jpg".

video with controls resized loops poster (.mp4)

This video element employs the controls attribute, giving the player some basic controls, is resized, loops, and has a poster.

The attributes used: controls width="848" height="480" loop poster="../../public/img/barbecue.jpg".
