object (application/x-shockwave-flash)

The object tag offers similar in functionality to the embed tag, and is typically used to embed non-HTML MIME type files. The MIME type is declared in the type attribute, and in this case, the MIME type is "application/x-shockwave-flash".

object resize

Objects can be resized with the attributes height and width. In this case, we set the attributes to: height="740" width="480".
