source for audio with controls (.m4a)

The following source element implementation replaces the src attribute for the audio element. It nests a source tag within the audio tag.

video with controls resized loops poster (.webm)

This video element has two nested source tags. The first tag gives the video a .webm source. If the client is unable to play that format, it will try the second .mov source.
