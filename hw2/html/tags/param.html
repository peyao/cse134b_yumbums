4.7.5 - param

index

param for object (quality="low")

The param tag serves to provide parameters to object elements. Param tags are nested within object tags.

In this example, we give the object plugin the parameter quality="low". This can be verified by right clicking the object and selecting "Quality" in the context menu.

param for object (quality="medium")

Just to verify, this time we're giving the object the parameter quality="medium".

param for object (quality="invalid-value")

If we give the object an invalid value, it will revert to its default value.
