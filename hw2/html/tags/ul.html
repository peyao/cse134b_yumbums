4.4 Grouping Content

This page covers the following elements:

  • 4.4.5 <ol>
  • 4.4.6 <ul>
  • 4.4.7 <li>
  • 4.4.8 <dl>
  • 4.4.9 <dt>
  • 4.4.10 <dd>

This page lists the following (content grouping) elements and give an example for each:

  • <ol>

    The ol is the ordered-list element. Below, I'm listing my favorite food in order then using the reversed attribute to reverse the list:

    My favorite food [in order]:
    1. Burger
    2. Pizza
    3. Salad

    My favorite food [in reverse order]:
    1. Salad
    2. Pizza
    3. Burger
  • <ul>

    The ul is the unordered-list element. This page is using an unordered list to list the html elements discussed:

  • <li>

    The li is the list-item element. Every item in the following list is added to the list using the li element:

    • red
    • green
    • blue
  • <dl>, <dt>, <dd>

    The dl is the description-list element. So if we have a list of words and their definitions, the list would be a dl element, the terms would be the dt elements, and the definitions would be the dd elements. Here's an example:

    Apple
    The round fruit of a tree of the rose family, which typically has thin red or green skin and crisp flesh. Also, Apple Inc. is the most valuable company in the world.
    crisp
    (Of a substance) firm, dry, and brittle, especially in a way considered pleasing or attractive
