4.7.7 - audio

index

audio (.m4a)

This is the most basic implementation of the audio element. It has no attributes, not even controls. This means you will have a hard time trying to start playing it, but the element still exists within this panel.

audio with autoplay controls (.m4a)

This is a simple audio element implementation, with controls.

In case you were wondering, this is the popular "Charlie Schmidt's Keyboard Cat! - THE ORIGINAL!" video from YouTube.

audio with controls loop muted (.m4a)

This is the audio element with the following attributes: controls loop muted.
