4.7.3 - embed

index

embed (video/quicktime)

The embed tag is typically used to embed non-HTML MIME type files. The MIME type is declared in the type attribute, and in this case, the MIME type is "video/quicktime".

The tag has no fallback content in case there are any compatibility issues. If the user does not have the correct plugin to the run the embedded file, then the embedded content simply fails to run.

embed resize (video/quicktime)

The embed tag can use the width and height attributes to customize the size of the embedded element.

embed (application/x-shockwave-flash)

A popular use of the embed tag is to load .swf files. Developers can pass in parameters to the plugin. In this case, the parameter quality is used to set the quality of the Flash player to "low".
