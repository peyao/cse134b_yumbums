base

The base element is used to specify a base URL. This means that any relative URLs on this document will use the base URL instead.

For example, the following link will have a relative URL reference that does not point to a page we have locally: Go to sections.html.

Note that the link above opens the document in a new tab, while our a tag does not have a target attribute. This is because we can set the target attribute in the base tag itself.
